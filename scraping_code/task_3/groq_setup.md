- ✅ **Super fast** - 100+ tokens/second

## Setup Steps (2 minutes)

### 1. Get API Key
1. Visit: https://console.groq.com/keys
2. Sign up with email (no payment info needed)
3. Click "Create API Key"
4. Copy your key

### 2. Add to Project
Run this command and paste your key:
