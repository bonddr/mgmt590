# 👗 Fashion Intelligence Assistant

Real-time fashion trend analysis powered by **Firecrawl API** and AI.

## 🚀 Features

- ✅ **Real-time scraping** from Pinterest, Zara, Uniqlo & Vogue
- ✅ **Lightning-fast** with Firecrawl (10x faster than Selenium)
- ✅ **AI-powered analysis** with brand customization
- ✅ **Beautiful UI** with Streamlit
- ✅ **Shared visual inspiration** (not brand-specific)
- ✅ **Downloadable reports** for each brand
